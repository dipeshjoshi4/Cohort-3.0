## Implementation Checklist
- [x] **Task Creation Module**: Build UI to input Task Title and select Category.
- [x] **Dynamic DOM Manipulation**: Create task cards using pure vanilla JS mechanisms (`createElement`, `appendChild`, etc.) without page refreshes.
- [x] **Attributes vs Properties**: Embed custom data attributes (`data-id`, `data-status`, `data-category`) on task card elements for state tracking.
- [x] **State Management**: Fully interactive interface allowing real-time status transitions and filters.
